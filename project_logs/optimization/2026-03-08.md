  - `app/main.py` 同步注册新增路由；整改文档中将 P3-1 标记为已完成。
- **涉及模块**：`app/routes/api_v1_posts.py`、`app/routes/api_v1_taxonomy.py`、`app/routes/api_v1_media.py`、`app/main.py`、`frontend/src/admin/pages/AdminPostsPage.jsx`、`frontend/src/admin/components/AdminPostEditor.jsx`、`frontend/src/admin/components/AdminPostList.jsx`、`docs/implementation-alignment-remediation-checklist.md`
- **实施方法**：采用最小拆分策略，仅抽离边界清晰的 taxonomy/media 路由与后台编辑子视图，不新增全局 store、不改变 API 契约、不扩大页面状态层级。
