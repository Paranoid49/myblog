- 依赖：T3-1, T2-2
- 验收：
  - 前台详情页不再直接使用原始文本输出正文

- 验收：

  - 文章列表从简单列表升级为卡片或列表块结构
  - light/dark 下都具备可读性

### T5-3 文章详情页阅读体验优化 ✅ 已完成
- 目标：让详情页达到可正常阅读博客文章的最低标准
