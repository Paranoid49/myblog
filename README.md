# myblog 最小可用版

## 快速启动

1. 配置环境变量（可选，默认读取 `.env`）
2. 一键启动（自动执行数据库迁移）

```bash
python scripts/start_blog.py
```

默认地址：`http://127.0.0.1:8000`

## 首次使用

1. 访问 `http://127.0.0.1:8000/setup` 完成初始化
2. 访问 `http://127.0.0.1:8000/admin/login` 登录后台
3. 在 `文章管理` 创建文章
4. 在 `分类与标签` 页面维护分类与标签

## 当前功能

- 首次初始化（管理员 + 站点信息）
- 管理员登录/退出
- 文章创建、编辑
- 文章发布 / 转为草稿
- 分类与标签创建
- 首页仅展示已发布文章
